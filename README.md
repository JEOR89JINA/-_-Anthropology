# 🔬 بوت الأنثروبولوجيا الذكي لتليجرام (Telegram Anthropology Bot)

بوت تليجرام ذكي ومتطور مصمم خصيصاً للعمل كخبير ومحلل في **علم الأنثروبولوجيا** (علم الإنسان). يقوم البوت بمراقبة منشورات القناة تلقائياً، وتحليلها باستخدام نموذج الذكاء الاصطناعي الخارق والسريع **Groq (Llama 3)**، ثم نشر التوضيحات والتحليلات العميقة مباشرة داخل مجموعة المناقشة التابعة للقناة، بالإضافة إلى الرد الذكي على تعليقات المشتركين وتساؤلاتهم.

---

## ✨ الميزات الأساسية
- **تحليل تلقائي لمنشورات القناة:** بمجرد النشر في القناة، يقوم البوت بصياغة تحليل أنثروبولوجي (ثقافي، اجتماعي، حيوي) ونشره في المجموعة.
- **التفاعل الذكي مع الأعضاء:** يرد البوت بذكاء وعمق عند الإشارة إليه (`@mention`) أو الرد على رسائله (`Reply`) أو عند ذكر كلمة "أنثروبولوجيا".
- **سرعة خارقة:** يعتمد البوت على واجهة برمجة تطبيقات **Groq API** لتقديم استجابات فورية ودقيقة.
- **آمن ومستقر:** مجهز بالكامل ليعمل على منصات الاستضافة السحابية مثل Railway أو Render على مدار 24 ساعة.

---

## 🛠️ الملفات المكونة للمشروع
1. `main.py`: الملف البرمجي الأساسي للبوت المكتوب بلغة بايثون.
2. `requirements.txt`: يضم المكتبات البرمجية اللازمة لتشغيل الكود (`pyTelegramBotAPI` و `groq`).
3. `Procfile`: ملف التكوين الخاص بمنصات الاستضافة لتحديد أمر تشغيل السيرفر.

---

## 🚀 طريقة الضبط والتشغيل

### 1. إعدادات تليجرام
1. قم بإنشاء بوت جديد عبر `BotFather@` واحتفظ بالتوكن السري (`TELEGRAM_TOKEN`).
2. قم بإضافة البوت **كمشرف (Admin)** في قناتك ومجموعتك المرتبطة بالقناة، وتأكد من تفعيل صلاحيات قراءة وإرسال الرسائل.

### 2. إعدادات منصة الاستضافة (Railway / Render)
عند ربط المستودع بمنصة الاستضافة، يجب إضافة متغير البيئة التالي في قسم **Variables** لحماية بياناتك:

| اسم المتغير (Key) | القيمة (Value) |
| :--- | :--- |
| `TELEGRAM_TOKEN` | ضع توكن البوت الخاص بك هنا من BotFather |

## 💻 التقنيات المستخدمة
- **Language:** Python 3
- **Framework:** pyTelegramBotAPI
- **AI Core:** Groq API (Llama-3-8b)
